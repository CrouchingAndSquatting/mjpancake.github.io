## 图文教程

1. 入门（必修）：[廖雪峰的Git教程][lxf]{:target="_blank"}
  （学习时长：一至三天）
2. 巩固补充（必修）：[Ruheng的Git教程][git-adv]{:target="_blank"}
  （学习时长：一至三天）
3. 深入（选修）：[Pro Git （中文版）][pro-git]{:target="_blank"}
  （学习时长：长期）<a name="_"></a>

## 教程怎么看

- 一个字一个字精读，每句话都看懂，不要半懂不懂
- 自己建个测试项目，动手实践一遍教程里的 Git 指令，不要产生光看就能学会的幻觉
- 遇到实在难以理解的部分，上网搜，问大佬
